# 🚩 Custom CTF Challenges

Welcome to my repository of custom-built Capture The Flag (CTF) challenges. 

I built this repository to showcase my ability to design, develop, and analyze complex systems from an offensive security perspective. By creating these challenges, I demonstrate a practical understanding of cryptography, algorithm design, and secure coding principles.

## 📂 Challenge Directory

| Challenge Name | Category | Description |
| :--- | :--- | :--- |
| **`lcg^2`** | Cryptography | A state-recovery puzzle exploiting nested Linear Congruential Generators (LCGs). |
| **`devil_cupid`** | Cryptography | A Mersenne Twister (MT19937) state-cloning attack disguised as a Valentine's dating sim. |
| **`intercepted_love_letters`** | Cryptography | An introductory AES-CTR nonce-reuse (Many-Time Pad) exploitation challenge. |

*(More challenges will be added as they are developed)*

## 🚀 Skills Demonstrated

Building these challenges required applying several advanced concepts:
* **Cryptography:** Vulnerability analysis of Pseudo-Random Number Generators (LCGs and MT19937 state-cloning), stream cipher vulnerabilities, and AES-CTR mode nonce-reuse (Many-Time Pad attacks).
* **Mathematics:** Applied modular arithmetic, modular inverses, calculating geometric series sums for efficient PRNG state-skipping, and XOR logic.
* **Software Engineering:** Object-oriented programming in Python, cleanly handling edge cases, and building interactive command-line interfaces.
* **Security Mindset:** Designing intentionally flawed logic that requires attackers to write custom exploit scripts to break.
